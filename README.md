# Coup

This is an online version of the card game, Coup!

Coup is a fast-paced card game of deception and strategy where players compete to eliminate their opponents. By bluffing, challenging, and using character abilities, youâ€™ll need to be clever and bold to be the last player with influence!  

Coup can be played with 2-6 players.  

Technologies used in the creation of this webapp:
* Frontend: Vite + React.js 
* Backend: Node.js/Express.js
* Database: Firebase
* Deployment: Render

## How to Run Locally
1. Clone the repository  
2. Open the project in a split terminal  
3. In one terminal, run:  
   `cd client/my-react-app`  
   In the other terminal, run:  
   `cd server`  
4. In both terminals, run:  
   `npm install`  
5. In the client-side terminal, run:  
   `npm run dev`  
   In the server-side terminal, run:  
   `npx nodemon server.js`
